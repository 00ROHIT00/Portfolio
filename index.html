Projects

The following are some of the projects that I've done in the past. Most of these are using HTML, CSS and and JavaScript. Some of these project also use PHP and MySQL.

  • CINEWISE
    (Movie Search Engine)

    • CineWise is a movie search engine. It was my BCA Final Year project. CINEWISE allows the user to search for any movies with their title, release year, rating, or even their genre. It basically used an database to search for the movies. The users can also register or sign into their accounts to save their watch later playlist. The Admins can see all the movies in the database, as well as edit/delete them. The user interface of the whole website is made using HTML, CSS and JavaScript. Mostly, HTML and CSS was used and hence it is a pretty lightweight website. I used Javascript as the frontend and backend. PHP and MySQL was used to store and retrieve the data about the movies, from the database.

  • Weather App
    (Using API)

    • The Weather App was made as an introduction to JavaScript and on how to use Application Programming Interfacee or APIs to get data from various sources. The API used for the project is the OpenWeatherMap API. The API provided many information like the current temperature, wind speed, wind direction, high temperatur, low temperature etc. The user could search for any city on the search box, and the inputted term would be forwarded to the API link along with my API key. We could then use the response from the API to acquire the real time termperature.
